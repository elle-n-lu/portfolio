About learning of basic Git Commands

Before creating a repo, create a SSH key or other authentication way, to avoid the authentication error when you push your project changes.

  1. Start with repo from github website

    In your repositories settings, create new repo, select add a README.md file and add a MIT license(the most used one).

    In the new repo, click the green button and clone the HTTPS link.

    Open Visual Studio Code, in the terminal area, execute

    git clone your-repo-link

  2. Already have a local project?

    You still need a repo created!

    In the terminal, navigate to your local prject folder, execute

    $ git init

    $ git add .

    $ git commit -m "first commit"

    $ git branch -M main

    $ git remote add origin 'your repo link'

    $ git pull 'your repo link' main

    $ git pull origin main --rebase

    $ git push -u origin main

  3. Other commans

    git status

    Check the current branch:

    git branch

    New Branch:

    git checkout -b <new-branch>

    Switch Branch:

    git checkout <branch>
